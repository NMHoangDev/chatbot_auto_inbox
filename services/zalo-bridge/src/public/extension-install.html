Markee Zalo Extension

Cài tiện ích để kết nối Zalo cá nhân với Markee Chat

Cài đặt một lần trên trình duyệt

Tải tiện ích Markee Zalo

Tiện ích này chỉ dùng để mở Zalo Web, lấy phiên đăng nhập của đúng tài khoản Zalo cá nhân và gửi về Markee Chat. Sau khi kết nối xong, người dùng thao tác trong Markee Chat.

Tải file cài đặt ZIP
1

Tải và giải nén ZIP

Tải file ZIP ở nút trên, sau đó giải nén ra một thư mục có tên dễ nhớ, ví dụ markee-zalo-extension.

2

Mở trang tiện ích Chrome

Mở Chrome, vào chrome://extensions, bật công tắc Developer mode ở góc trên bên phải.

3

Chọn Load unpacked

Bấm Load unpacked và chọn đúng thư mục đã giải nén ở bước 1.

4

Quay lại Markee Chat

Tải lại trang kết nối Zalo cá nhân trong Markee Chat, rồi bấm Đăng nhập tài khoản Zalo.

Bản cài đặt nội bộ này dùng cho giai đoạn test. Để khách hàng không cần bật Developer mode, nên publish tiện ích lên Chrome Web Store và cấu hình Markee trỏ đến link cài đặt chính thức.
